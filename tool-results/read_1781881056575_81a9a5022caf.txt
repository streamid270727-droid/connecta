    55→- Created /api/messages/route.ts (POST, body { conversationId, content }): validates content non-empty + max 2000 chars (Indonesian errors), validates conversation exists + user is participant (403), derives recipientId as the OTHER user, creates DirectMessage (isRead=false), bumps conversation.updatedAt, upserts MessageRead for the sender so sender has no unread, then best-effort emits dm:message (with senderName/content/createdAt) and notifyUser("Pesan baru", truncated body) to recipient, and persists a Notification row (type "message", content "mengirimi Anda pesan", entityId=conversationId). Returns the created message with sender.
    56→- Created /api/messages/[id]/read/route.ts (POST, params.id = messageId): loads message, 404 if missing, 403 if recipientId !== current user, otherwise flips isRead=true if not already read and returns { success: true }.
